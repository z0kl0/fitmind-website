# FitMind Website

## Description
This is the official marketing and informational website for **FitMind Haiti**. The project is built as a responsive, multi-page static site designed to provide users with resources on the company. FitMind is a human centered coaching and training company founded in 2019, focused on creating meaningful and lasting change by working at the level of mindset. Through corporate seminars, business assessments, wellness events, and speaking engagements, they support individuals and organizations in building clarity, alignment, and sustainable growth. Guided by the principle Shape your mind. Change your life.

The site currently features:
* **Home (`index.html`):** The main landing page with core value propositions.
* **About (`about.html`):** Detailed background on the FitMind mission and team.

## Tech Stack
* **Core:** HTML5 (Semantic)
* **Styling:** CSS3 (Custom properties, Flexbox/Grid)
* **Assets:** Organized resource management via the `assets/` directory.
